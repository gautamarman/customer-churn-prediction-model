# Customer Churn Prediction using R

## Project Overview
This project focuses on predicting customer churn using Machine Learning in R. The objective was to identify customers likely to leave a service based on customer demographics, subscription details, and billing information.

The project includes:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Logistic Regression Model
- Churn Prediction
- Business Insights & Recommendations

---

# Objective
To analyze customer behavior patterns and build a predictive model that helps businesses identify customers at risk of churning.

---

# Project Workflow

## 1. Data Cleaning
- Removed unnecessary columns
- Handled missing values
- Converted data types
- Processed categorical variables

## 2. Exploratory Data Analysis
Analyzed:
- Churn distribution
- Monthly charges
- Contract type
- Customer tenure
- Payment methods

## 3. Machine Learning
Built a Logistic Regression model to predict customer churn.

## 4. Model Evaluation
Evaluated model performance using:
- Confusion Matrix
- Accuracy
- Precision
- Recall

---

# Key Business Insights

- Customers with month-to-month contracts showed significantly higher churn rates.
- Higher monthly charges were associated with increased churn probability.
- Long-tenure customers were more likely to stay loyal.
- Customers using electronic check payments demonstrated higher churn behavior.

---

# Business Recommendations

- Introduce loyalty programs for short-term customers.
- Offer personalized discounts for high-risk customers.
- Improve onboarding and engagement strategies for new users.
- Encourage customers to switch to long-term subscription plans.
